Program is responsible for collecting pit reports from multi module project, copy them to one directory and aggregate index.html files to one.

Program has three parameters:
- projectRootPath - path to project root (required),
- concatDirectoryPath - path to directory where all report will be copied and aggregated index.html will be created (required),
- pitDirName - directory name for pit reports defined in pitest plugin, default is pit-reports (optional)
